This component is used to calculate the overhaul progress of the growth of the plante.

This include the number of leaves (or leaf layers) and the phase

The phase is a number going from 0 to 7. The phase is what is used to know when the simulation must ends (phase > 6) in SQ.

This component also contains a calendar which is used to stored triplets (date, cumulTT, zadok stage) for each zadok stage. Most of those triplets are stored during the actual process of this component but not all of them.
